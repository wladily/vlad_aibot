# vlad_aibot

> Умный Telegram-бот основанный на ИИ, который помогает пользователям разобраться в айти и программировании, а также исправить код.

### Основные возможности

- Команда `/start` - старт работы
- Команда `/ai` - задать вопрос ии-помощнику
- Команда `/fix` - исправить код (рефакторинг и отладка)
- Кнопка `AI-помощник` - задать вопрос ии-помощнику
- Кнопка `Исправление кода` - исправить код (рефакторинг и отладка)
- Кнопка `Справка` - старт работы
- Кнопка `Исправление ошибок` - отладка кода
- Кнопка `Улучшение кода` - рефакторинг кода

## Технологии

- TypeScript
- Libraries: OpenAi, Grammy

## Установка зависимостей

```bash
npm install
```

## Запуск проекта в режиме разработки

```bash
npm run dev
```

## Сборка проекта (production)

```bash
npm run build
```
Результат будет в папке deploy/
